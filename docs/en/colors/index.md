# Primary Color — BAUER Orange

The primary color of BAUER GROUP is **Orange (#FF8500)**. It stands for energy, innovation, and warmth and is the central identity feature of the brand.

## BAUER Orange — #FF8500

  #FF8500

| Format | Value | Authority |
|--------|-------|-----------|
| **HEX** | `#FF8500` | Print, SVG, tooling fallback |
| **OKLCH** | `oklch(68% 0.19 47)` | Web, display, wide-gamut |
| **RGB** | `255, 133, 0` | — |
| **HSL** | `31°, 100%, 50%` | — |
| **CMYK** | `C0 M48 Y100 K0` | Print (calibrated) |
| **Pantone** | `PMS 144 C` (Coated) / `PMS 144 U` (Uncoated) | Spot color print |

::: info Notation hierarchy
**Print** (printed materials, business cards, Pantone conversion): the **HEX/CMYK/Pantone values** are authoritative. Print shops convert HEX → CMYK via calibrated profiles.
**Web/display** (apps, sites, P3, HDR): the **OKLCH values** give the correct perceptive impression on modern displays. HEX remains a fallback for tooling that doesn't yet understand OKLCH.
**SVG/logo, Adobe/Figma tooling**: **HEX only** (for maximum compatibility).

→ [Token methodology & conventions](/en/colors/methodology)
:::

### As text color

::: warning Accessibility
Orange 500 (#FF8500) has a **contrast ratio of only 3.1:1** on white — that's **not enough** for normal text (WCAG AA requires 4.5:1).

For text, use these instead:

- <span style="color:#C2570A;font-weight:700;font-size:1.1em;">Orange 700 (#C2570A)</span> — 4.6:1 contrast → AA compliant
- <span style="color:#9A4509;font-weight:700;font-size:1.1em;">Orange 800 (#9A4509)</span> — 6.4:1 contrast → AA+ compliant
:::

## Supporting Colors — Warm Grays

BAUER GROUP follows a **monochromatic color scheme**: orange is the only chromatic color. All supporting tones are based on warm-tinted grays (hue ≈28°), which harmonize naturally with orange.
