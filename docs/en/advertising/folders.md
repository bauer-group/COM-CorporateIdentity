# Folders

BAUER GROUP uses two folder formats: **DIN-Lang** (with closure flap) and **DIN-A4** (with business card slot). Both are professionally designed with photo motifs, logo, and QR code.

## Specifications

| Property | DIN-Lang | DIN-A4 |
|----------|----------|--------|
| **Format** | DIN-Lang, landscape (#51) | DIN-A4, portrait |
| **Features** | Closure flap | Business card slot |
| **Paper** | 300 g/m² art paper, matte coated | 300 g/m² art paper, matte coated |
| **Finishing** | Glossy cello, single-sided | Glossy cello, single-sided |
| **Printing** | Four-color (4/0), back unprinted | Four-color, both sides (4/4) |
| **Color profile** | ISO 12647-2 (PSO) | ISO 12647-2 (PSO) |
| **Data check** | Yes | Yes |
| **Order quantity** | 1,000 units | 1,000 units |
| **Version** | 2023-07-14 | 2023-06-26 |

## Design

### Common Elements

| Element | Details |
|---------|---------|
| **Photo motif** | Large-area image motif (office/team scenes) as background |
| **Logo** | BAUER GROUP logo (white on black) in the lower area |
| **Website** | `www.bauer-group.com` (white on black bar) |
| **QR code** | Product-specific QR code |
| **Orange accent** | Orange vertical bar (DIN-Lang) or pocket flap (A4) |
| **Gray accent** | Gray area accent as counterpoint |

### QR Codes

| Product | QR code URL | Shortlink ID |
|---------|-------------|--------------|
| **DIN-Lang folder** | `https://go.bauer-group.com/p-folderdl` | `p-folderdl` |
| **DIN-A4 folder** | `https://go.bauer-group.com/p-foldera4` | `p-foldera4` |

## Print Specification

| Property | Value |
|----------|-------|
| **Color mode** | CMYK (process colors) |
| **Color profile** | ISO 12647-2 (PSO Coated v3) |
| **Orange** | C0 M48 Y100 K0 / PMS 144 C |
| **Bleed** | 3 mm all around |
| **Photo resolution** | Min. 300 dpi at final format |
