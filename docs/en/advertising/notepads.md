## Layout Structure

Both formats follow exactly the same layout — scaled to the respective sheet size.

### Header Area

<div style="background:var(--vp-c-bg-soft);border-radius:12px;padding:8px;margin:16px 0;">
  <img src="/print/block-a5-header.png" alt="Notepad header area" style="width:100%;border-radius:4px;">
</div>

| Element | Position | Details |
|---------|----------|---------|
| **Logo** | Top left | Wide variant with symbol (orange diamond + "B"), full-color on white |
| **QR code + CTA** | Top right | QR code (black on white) with call-to-action: info icon (ℹ speech bubble + arrow icon in orange) |
| **Ruler** | Left edge | Centimeter scale (0–20 cm for A5, 0–14 cm for A6), tick marks and numbers in BAUER Orange |

### Writing Area

| Element | Details |
|---------|---------|
| **Grid** | Graph paper print: 5 mm checkered grid slightly darker, mm lines lighter (light gray). Begins below the header area |
| **Watermark** | "B" symbol, in the lower third of the writing area, right-aligned, 5–10% opacity (analogous to letterhead) |
| **Grid area** | Extends from the header area to the footer area, from the ruler edge to the right sheet edge |

## QR Codes

Each notepad has an individual QR code with a product-specific URL via the BAUER GROUP URL shortener.

| Product | QR code URL | Shortlink ID |
|---------|-------------|--------------|
| **Notepad A5** | `https://go.bauer-group.com/p-pada5` | `p-pada5` |
| **Notepad A6** | `https://go.bauer-group.com/p-pada6` | `p-pada6` |

| Property | Value |
|----------|-------|
| **URL shortener** | `go.bauer-group.com` |
| **Naming scheme** | `p-pad{format}` (p = print, pad = notepad) |
| **Size A5** | approx. 15 × 15 mm |
| **Size A6** | approx. 12 × 12 mm |
| **Position** | Top right, flush with logo top edge |
| **Color** | Black on white |

::: tip URL shortener advantage
The shortener `go.bauer-group.com` decouples the print product from the target URL. The redirect can be changed server-side at any time — without reprinting. Each product receives its own trackable shortlink ID.
:::

### QR Code Rules (all print materials)

1. **Minimum size**: 10 × 10 mm (for reliable scanning)
2. **Quiet zone**: At least 2 mm white margin around the QR code
3. **Color**: Black on white — no orange, no inverted background
4. **Error correction**: Level M (15%) or higher
5. **Icon companion**: Always accompanied by an info or scan icon
6. **URL**: Always use `go.bauer-group.com/...`, no direct long URLs
