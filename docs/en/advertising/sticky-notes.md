## Specifications

| Property | Value |
|----------|-------|
| **Format** | Square, 72 × 72 mm |
| **Paper** | 80 g/m² offset paper |
| **Sheets** | 50 sheets |
| **Binding** | Glued at the head |
| **Printing** | Four-color (4/0) |
| **Data check** | None |
| **Order quantity** | 1,000 units |
| **Version** | 2023-06-26 |

## Layout Structure

| Element | Position | Details |
|---------|----------|---------|
| **Logo** | Top left | Wide variant with symbol, scaled small |
| **QR code + CTA** | Top right | Info icon (ℹ speech bubble + arrow in orange) + QR code |
| **Watermark** | Centered | "B" symbol, large, ~5–10% opacity |
| **Website** | Bottom right | `www.bauer-group.com` |
| **Accent bar** | Bottom edge | BAUER Orange (#FF8500) with gray closure on the right |

::: tip Design principle
The sticky notes use exactly the same design system as the notepads — logo top left, QR+CTA top right, watermark centered, accent bar at the bottom. No grid and no ruler due to the smaller format.
:::
