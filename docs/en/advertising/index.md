## Logo Color Specification (Print)

Binding color values for all print materials:

| Color | CMYK / Spot color | RGB (screen) | Note |
|-------|-------------------|--------------|------|
| **Orange** | PMS 144 C/U | `#FF8500` | Primary color, symbol |
| **White** | Solid white | `#FFFFFF` | RAL 9016 (traffic white) |
| **Black** | Solid black | `#000000` | Wordmark "BAUER" |
| **Gray** | PMS 419 C/U | `#2F3234` | RAL 7021 (black gray), wordmark "GROUP" |

::: warning Logo frame
The frame around the logo should be kept narrower — about ½ to ¼ of the standard width (hairline). For details, see the print templates.
:::

## Phone Number Format

Area codes are displayed **without parentheses**. Applies uniformly to all business stationery.

| Format | Example |
|--------|---------|
| **Phone** | Tel. +49 9971 20098 - 199 |
| **Fax** | Fax +49 9971 20098 - 4199 |
| **Mobile** | Mobile +49 178 1002005 |

## General Print Guidelines

All print materials follow the specifications from [Print & Production](/en/print/):

- Colors: use CMYK values, no RGB
- Spot color: Pantone PMS 144 C (where possible)
- Paper: see [paper recommendations](/en/print/paper)
- QR codes: always via `go.bauer-group.com` — see [QR code rules](/en/advertising/notepads#qr-code-rules-all-print-materials)
