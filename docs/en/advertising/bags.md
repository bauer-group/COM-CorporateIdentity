## Specifications

| Property | Value |
|----------|-------|
| **Type** | Cotton bag |
| **Color** | White |
| **Print area** | 200 × 200 mm |
| **Print colors** | 1 color (orange) |
| **Printing** | Four-color (4/0) |
| **Order quantity** | 250 units |
| **Version** | 2023-06-25 |

## Design

| Element | Details |
|---------|---------|
| **Motif** | "B" symbol (diamond with "B"), full-area at 200 × 200 mm |
| **Color** | BAUER Orange (PMS 144 C) |
| **Orientation** | Slightly rotated (as in the logo standard) |

::: tip Design principle
The bag relies exclusively on the symbol — without wordmark or URL. The large, eye-catching presentation creates high recognition value.
:::
