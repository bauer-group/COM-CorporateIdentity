## Core Principles

- **Consistency** — Uniform appearance across all digital channels
- **Primary color sparingly** — Orange (#FF8500) as accent, not as background
- **Readability** — Maintain WCAG 2.1 AA contrast in digital media as well
- **Responsive** — Templates must work on all screen sizes
