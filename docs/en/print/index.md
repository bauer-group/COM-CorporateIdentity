# Print & CMYK

Color specifications for professional print production.

## CMYK Primary Colors

| Color | CMYK | Application |
|-------|------|-------------|
| **BAUER Orange** | C0 M48 Y100 K0 | Primary color, logo icon |
| **Black** | C0 M0 Y0 K100 | Headlines, logo wordmark |
| **Gray** | C0 M0 Y0 K65 | Secondary text, "GROUP" |

## Pantone Color References

| Color | Coated | Uncoated |
|-------|--------|----------|
| **Orange** | PMS 144 C | PMS 144 U |
| **Dark Orange** | PMS 1525 C | — |

::: tip Pantone usage
For spot color print (e.g., business cards, letterhead) always use the **Pantone reference**. CMYK values are approximations and may vary depending on the print process.
:::

## Print Notes

### Logo Print

- Minimum size Wide Logo: **40mm** width
- Minimum size Square Logo: **12mm** width
- Always supply the logo as **vector files** (SVG, EPS)
- Do not use JPG/PNG for print

### Color Accuracy

- Request a proof print before production runs
- Use ICC profiles (ISO Coated v2 for offset)
- Enable overprint for black
