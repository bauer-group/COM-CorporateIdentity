## Design

| Element | Position | Details |
|---------|----------|---------|
| **Logo** | Top, centered | Icon + wordmark "BAUER GROUP" (black) |
| **Address** | Center | Janahof 30 · D-93413 · Cham (Germany) |
| **Phone** | Below address | Tel. +49 (9971) 20098 - 0 * |
| **Fax** | Below phone | Fax +49 (9971) 20098 - 400 * |
| **URL** | Bottom, centered | `www.bauer-group.com` |

\* Current print uses parentheses around the area code. On reprint: area codes without parentheses (see [Business Cards → Phone Number Format](/en/print/business-cards#phone-number-format)).

::: tip Design principle
The stamp is deliberately reduced to the essentials — logo, address, and contact information. Single-color black for universal use on any paper type.
:::
