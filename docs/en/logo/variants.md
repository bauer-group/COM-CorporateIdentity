# Logo variants

## Color variants

### Standard — Light background

The standard variant for all applications on a light background.

<div class="logo-stage light">
  <img src="/brand/bauer-group-logo-wide.svg" alt="BAUER GROUP Logo Standard" style="max-width: 380px" />
</div>

| Element | Color |
|---------|-------|
| **Icon** | Orange (#FF8500) + white "B" |
| **BAUER** | Dark (#231F1F) |
| **GROUP** | Gray (#52525B) |

### On dark — Dark background

On a dark background, the type is rendered in white. The icon remains unchanged.

<div class="logo-stage dark">
  <img src="/brand/bauer-group-logo-wide-white.svg" alt="BAUER GROUP Logo on dark" style="max-width: 380px" />
</div>

| Element | Color |
|---------|-------|
| **Icon** | Orange (#FF8500) + white "B" (unchanged) |
| **BAUER** | White (#FFFFFF) |
| **GROUP** | Light gray (#D4D4D8) |

### Icon (Square)

The icon alone is used when space is limited.

::: tip Note
The icon (orange diamond with white B) does **not** change — it works the same on light and dark backgrounds.
:::
