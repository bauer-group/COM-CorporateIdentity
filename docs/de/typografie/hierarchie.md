# Typografische Hierarchie

## Skalierung

| Ebene | Gewicht | Größe | Letter-Spacing | Anwendung |
|-------|---------|-------|----------------|-----------|
| **Display** | 700 (Bold) | 44px | -0.02em | Heldenbereich, Landingpages |
| **H1** | 700 (Bold) | 32px | -0.01em | Seitentitel |
| **H2** | 600 (Semibold) | 24px | — | Abschnittstitel |
| **H3** | 600 (Semibold) | 18px | — | Unterabschnitte |
| **Body** | 400 (Regular) | 16px | — | Fließtext |
| **Small** | 400 (Regular) | 14px | — | Beschriftungen, Fußnoten |

## Schriftgewichte

| Gewicht | Wert | Verwendung |
|---------|------|-----------|
| **Regular** | 400 | Fließtext, Beschreibungen |
| **Semibold** | 600 | Abschnittstitel, Labels |
| **Bold** | 700 | Seitentitel, Hervorhebungen, Display |

::: tip Regel
Verwenden Sie maximal **2 Gewichte** auf einer Seite. Die Kombination Regular + Bold reicht für die meisten Anwendungen aus.
:::
