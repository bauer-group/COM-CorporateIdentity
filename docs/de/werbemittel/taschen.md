| Element | Details |
|---------|---------|
| **Motiv** | "B"-Bildmarke (Raute mit "B"), flächenfüllend auf 200 × 200 mm |
| **Farbe** | BAUER Orange (PMS 144 C) |
| **Ausrichtung** | Leicht gedreht (wie im Logo-Standard) |

::: tip Gestaltungsprinzip
Die Tasche setzt ausschließlich auf die Bildmarke — ohne Wortmarke oder URL. Die große, plakative Darstellung erzeugt hohen Wiedererkennungswert.
:::
