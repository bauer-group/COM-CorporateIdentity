# Print & CMYK

Farbspezifikationen für die professionelle Druckproduktion.

## CMYK Primärfarben

| Farbe | CMYK | Anwendung |
|-------|------|-----------|
| **BAUER Orange** | C0 M48 Y100 K0 | Primärfarbe, Logo-Bildmarke |
| **Schwarz** | C0 M0 Y0 K100 | Headlines, Logo-Wortmarke |
| **Grau** | C0 M0 Y0 K65 | Sekundärtext, „GROUP" |

## Pantone Farbreferenzen

| Farbe | Coated | Uncoated |
|-------|--------|----------|
| **Orange** | PMS 144 C | PMS 144 U |
| **Dunkelorange** | PMS 1525 C | — |

::: tip Pantone-Verwendung
Für Sonderfarben-Druck (z.B. Visitenkarten, Briefpapier) immer die **Pantone-Referenz** verwenden. CMYK-Werte sind Approximationen und können je nach Druckverfahren variieren.
:::

## Druckhinweise

### Logo-Druck

- Mindestgröße Wide Logo: **40mm** Breite
- Mindestgröße Square Logo: **12mm** Breite
- Logo immer in **Vektordateien** (SVG, EPS) liefern
- Keine JPG/PNG für Print verwenden

### Farbgenauigkeit

- Proof-Druck vor der Serienproduktion anfordern
- ICC-Profile verwenden (ISO Coated v2 für Offset)
- Überdrucken für Schwarz aktivieren
