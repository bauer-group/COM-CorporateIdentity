# Papierempfehlungen

## Geschäftsausstattung

| Produkt | Grammatur | Papierart | Hinweis |
|---------|-----------|-----------|---------|
| **Visitenkarten** | 350 g/m² | Matt gestrichen | Soft-Touch-Laminierung für Premium-Haptik |
| **Briefpapier** | 100–120 g/m² | Offset, naturweiß | Optimale Schriftdarstellung |
| **Umschläge** | 100 g/m² | Offset, naturweiß | Passend zum Briefpapier |
| **Briefbogen 2. Seite** | 90 g/m² | Offset, naturweiß | Leichteres Papier für Folgeseiten |

## Marketingmaterial

| Produkt | Grammatur | Papierart | Hinweis |
|---------|-----------|-----------|---------|
| **Broschüren (Cover)** | 250–300 g/m² | Bilderdruck glänzend | Matt-Laminierung optional |
| **Broschüren (Inhalt)** | 170 g/m² | Bilderdruck | Glänzend für Fotos, matt für Text |
| **Flyer** | 170–250 g/m² | Bilderdruck | Je nach Einsatzzweck |
| **Poster** | 170 g/m² | Bilderdruck glänzend | Ab A2 auch 135 g/m² möglich |

## Veredlungen

| Veredelung | Anwendung |
|-----------|-----------|
| **Soft-Touch-Laminierung** | Visitenkarten, Ordnerdeckblätter |
| **UV-Spotlack** | Logo-Hervorhebung auf Visitenkarten |
| **Blindprägung** | Premium-Briefpapier |
| **Folienprägung (Orange)** | Sonderanfertigungen |

::: info Nachhaltigkeitshinweis
Bevorzugen Sie **FSC-zertifiziertes** oder **Recycling-Papier**, sofern die Druckqualität nicht beeinträchtigt wird. Für Imagematerialien ist hochwertiges Primärfaser-Papier akzeptabel.
:::
