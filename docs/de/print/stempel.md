## Gestaltung

| Element | Position | Details |
|---------|----------|---------|
| **Logo** | Oben, zentriert | Bildmarke + Wortmarke "BAUER GROUP" (Schwarz) |
| **Adresse** | Mitte | Janahof 30 · D-93413 · Cham (Deutschland) |
| **Telefon** | Unter Adresse | Tel. +49 (9971) 20098 - 0 * |
| **Fax** | Unter Telefon | Fax +49 (9971) 20098 - 400 * |
| **URL** | Unten, zentriert | `www.bauer-group.com` |

\* Aktueller Druck verwendet Klammern um die Vorwahl. Bei Neudruck: Vorwahlen ohne Klammern (siehe [Visitenkarten → Telefonnummern-Format](/de/print/visitenkarten#telefonnummern-format)).

::: tip Gestaltungsprinzip
Der Stempel ist bewusst auf das Wesentliche reduziert — Logo, Adresse und Erreichbarkeit. Einfarbig Schwarz für universellen Einsatz auf allen Papierarten.
:::
