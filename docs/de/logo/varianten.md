# Logo-Varianten

## Farbvarianten

### Standard — Heller Hintergrund

Die Standardvariante für alle Anwendungen auf hellem Hintergrund.

<div class="logo-stage light">
  <img src="/logo-wide.svg" alt="BAUER GROUP Logo Standard" style="max-width: 380px" />
</div>

| Element | Farbe |
|---------|-------|
| **Bildmarke** | Orange (#FF8500) + Weiß „B" |
| **BAUER** | Dunkel (#231F1F) |
| **GROUP** | Grau (#52525B) |

### Auf Dunkel — Dunkler Hintergrund

Auf dunklem Hintergrund wird die Schrift in Weiß dargestellt. Die Bildmarke bleibt unverändert.

<div class="logo-stage dark">
  <img src="/logo-wide-white.svg" alt="BAUER GROUP Logo auf Dunkel" style="max-width: 380px" />
</div>

| Element | Farbe |
|---------|-------|
| **Bildmarke** | Orange (#FF8500) + Weiß „B" (unverändert) |
| **BAUER** | Weiß (#FFFFFF) |
| **GROUP** | Hell-Grau (#D4D4D8) |

### Bildmarke (Square)

Die Bildmarke allein wird verwendet, wenn der Platz begrenzt ist.

    <img src="/logo.svg" alt="Bildmarke auf Dunkel" style="max-width:80px" />
  </div>
</div>

::: tip Hinweis
Die Bildmarke (oranges Diamant mit weißem B) ändert sich **nicht** — sie funktioniert auf hellen und dunklen Hintergründen gleich.
:::
