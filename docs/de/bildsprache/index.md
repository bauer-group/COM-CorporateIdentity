# Bildsprache & Fotostil

::: warning In Arbeit
Dieser Abschnitt wird derzeit ausgearbeitet. Moodboards und Beispielbilder folgen in Kürze.
:::

Die Bildsprache der BAUER GROUP transportiert **Professionalität, Nahbarkeit und Technologiekompetenz**. Bilder unterstützen die Markenbotschaft — sie sind kein Lückenfüller.

## Stilrichtung

| Eigenschaft | So verwenden | So NICHT verwenden |
|-------------|-------------|-------------------|
| **Authentisch** | Echte Arbeitsumgebungen, echte Menschen | Gestellte Stockfotos, übertrieben lächelnd |
| **Professionell** | Klare Belichtung, ruhige Komposition | Überbearbeitet, HDR-Effekte, Filter |
| **Technologisch** | Moderne Arbeitsplätze, Screens, Code | Veraltete Hardware, Clip-Art-Stil |
| **Warm** | Natürliches Licht, warme Farbtöne | Kalte, sterile Büro-Atmosphäre |

## Farbbehandlung

- **Weißabgleich:** Leicht warm (passend zum Orange der Marke)
- **Sättigung:** Natürlich, nicht übersättigt
- **Kontrast:** Moderat — keine extremen Schatten oder Überbelichtung
- **Farbfilter:** Keine Instagram-artigen Filter

## Bildkomposition

- **Goldener Schnitt** oder Drittel-Regel für Hauptmotive
- **Weißraum** für Textüberlagerung einplanen
- **Blickrichtung** von Personen ins Bild hinein, nicht heraus
- **Tiefenunschärfe** für Fokus auf Hauptmotiv

## Geplante Inhalte

- Do/Don't-Galerie mit kommentierten Beispielen
- Stockfoto-Richtlinien (empfohlene Anbieter, Suchbegriffe)
