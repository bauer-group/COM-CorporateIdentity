### Logo Square (Quadratisch)

| Format | Download | Anwendung |
|--------|----------|-----------|
| SVG | [bauer-group-logo-square.svg](/downloads/bauer-group-logo-square.svg) | Favicons, App-Icons, Social Media |
| PNG | [bauer-group-logo-square.png](/downloads/bauer-group-logo-square.png) | Fallback, wenn kein SVG möglich |
| EPS | [bauer-group-logo-square.eps](/downloads/bauer-group-logo-square.eps) | Druckproduktion, Vektorgrafik |

::: tip Download-Hinweis
Für **digitale Anwendungen** verwenden Sie SVG-Dateien. Für **Druckproduktion** verwenden Sie EPS-Dateien. PNG nur als Fallback, wenn Vektorformate nicht unterstützt werden.
:::
