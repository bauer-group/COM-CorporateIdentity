# Digitale Medien

Richtlinien für die konsistente Markendarstellung in digitalen Kanälen — Social Media, Web und E-Mail.

## Übersicht

| Kanal | Formate | Status |
|-------|---------|--------|
| Social Media | Beiträge, Stories, Banner, Profilbilder | In Arbeit |
| Web | Header, Banner, Open Graph Images | In Arbeit |
| E-Mail | Signaturen, Newsletter-Templates | In Arbeit |

## Grundprinzipien

- **Konsistenz** — Einheitliches Erscheinungsbild über alle digitalen Kanäle
- **Primärfarbe sparsam** — Orange (#FF8500) als Akzent, nicht als Fläche
- **Lesbarkeit** — WCAG 2.1 AA Kontrast auch in digitalen Medien einhalten
- **Responsive** — Vorlagen müssen auf allen Bildschirmgrößen funktionieren
